\### 1. Агрегація даних через API

Написав Python-скрипт, який автоматично витягує сирі дані (Raw Data) по API з Google Ads, Facebook Graph API та платформи Shopify.



\### 2. Очищення та злиття (Data Wrangling)

crm\_data = pd.read\_csv('export\_crm.csv')

merged\_df = pd.merge(ads\_data, crm\_data, on=\['date', 'utm\_campaign'], how='left')

merged\_df\['real\_roas'] = (merged\_df\['crm\_revenue'] / merged\_df\['spend']) \* 100



\# Пошук найприбутковіших днів тижня

insights = merged\_df.groupby('day\_of\_week')\['real\_roas'].mean().sort\_values(ascending=False)

3. Налаштування Telegram-алертів
Додав до скрипта функцію відправки сповіщень. Тепер, якщо ROAS певної кампанії падає нижче критичного рівня (наприклад, <150%) протягом 24 годин, скрипт автоматично надсилає сповіщення в робочий Telegram-чат.
